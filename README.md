# 贝贝可可大冒险 (Beibei & Keke's Adventure)

一个基于 React + Vite + Tailwind CSS 开发的经典导弹拦截塔防游戏。

## 游戏特性
- **经典玩法**：拦截敌方火箭，保护城市和炮台。
- **多样化敌人**：三种不同类型的外星人（普通、快速、坦克）。
- **资源管理**：有限的弹药储备，每轮结束后自动补充。
- **视觉效果**：浩瀚星空背景，卡通风格的外星人设计。
- **音效系统**：击中敌人与被击中时有不同的音效反馈。
- **双语支持**：支持中英文切换。

## 在线演示
你可以将此项目部署到 Vercel 或 Netlify。

## 本地开发

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 部署到 Vercel
1. 将代码上传到 GitHub。
2. 在 Vercel 中导入该仓库。
3. Vercel 会自动识别 Vite 项目并进行部署。

## 技术栈
- **React 19**
- **Vite 6**
- **Tailwind CSS 4**
- **Motion (Framer Motion)**
- **Lucide React** (图标)
